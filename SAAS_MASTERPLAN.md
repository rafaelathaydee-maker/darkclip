# DarkTrend — SaaS Masterplan

## Vision
The go-to tool for dark-page creators who want to find viral content before it explodes.
"Netflix for viral intelligence."

## Phase 1 — Frontend Foundation (CURRENT)
- Next.js app shell + design system
- Homepage feed (5 sections)
- Niche tab filtering
- Video card system with hover states
- Mock data (30+ videos, 10 niches)
- No auth, no API

## Phase 2 — Auth + Collections
- Supabase auth (email + OAuth)
- Save videos to collections
- User profile + history
- Onboarding flow

## Phase 3 — Real Data
- YouTube Shorts API integration
- Trend scoring algorithm (views/time, engagement rate)
- Viral score calculation
- Auto-refresh feed

## Phase 4 — Deep Analytics
- Per-niche trend pages
- Growth curve charts
- Format analysis (what's working)
- Competitor niche mapping

## Phase 5 — Monetization
- Free tier: 5 saves/day, limited niches
- Pro tier: Unlimited + downloads + early signals
- Elite tier: API access + custom alerts + team seats

## North Star Metric
Daily Active Creators (DACs) who discovered a viral idea via DarkTrend that week.

## Positioning
- NOT a social media scheduler
- NOT a generic trend tracker
- A secret weapon for dark-page creators who understand the algorithm
