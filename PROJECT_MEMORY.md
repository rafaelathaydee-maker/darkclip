# DarkTrend — Project Memory

## Product
**DarkTrend** is a premium viral-content discovery platform for dark-page creators.
The feeling: addictive, secretive, futuristic, premium, cinematic.

## Purpose
Help creators find viral clips, fast-growing trends, high-retention content, niche opportunities — before anyone else does.

## Stack
- **Frontend:** Next.js 16 + TypeScript + Tailwind CSS v4 + Framer Motion 12
- **Backend:** Next.js API routes
- **Database:** Supabase (Phase 2+)
- **Hosting:** Vercel
- **Data:** Mock data (Phase 1) → YouTube Shorts API (Phase 3)

## Design DNA
- Base: near-black violet-tinted surface (`oklch(7% 0.008 280)`)
- Accent: electric violet (`oklch(72% 0.28 295)`) — secretive, not overused
- Typography: Geist Sans + Geist Mono (numbers/scores)
- Motion: Emil Kowalski principles — ease-out, sub-300ms, no scale(0), no bounce

## Anti-References
- Generic SaaS dashboards (Linear-light, pale blue)
- Neon-on-black crypto/web3 (too obvious)
- Glassmorphism as default (purposeful only: navbar, tab bar, card hover)
- Hero metric templates
- Identical card grids
